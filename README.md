Cooperative MAC Protocols for High-Throughput Wireless Sensor Networks and Human-Centric XAI Frameworks for IoT Security Monitoring together address both efficient data communication and intelligent, transparent security in modern IoT systems. Cooperative MAC protocols improve network performance by enabling sensor nodes to collaborate in transmitting data, reducing collisions, saving energy, and increasing throughput in wireless sensor networks. At the same time, human-centric XAI frameworks enhance IoT security by using explainable artificial intelligence to detect threats and clearly communicate the reasons behind security decisions to users. This combination ensures that IoT systems are not only fast, reliable, and energy-efficient in handling large-scale data communication but also secure, trustworthy, and easy to manage, as users can understand and respond effectively to potential risks.
